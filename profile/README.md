## 📖 关于我们

物化技小组（Physics Chemistry and Technology Group, P.C.T.G.）由河北涿州中学学子于 **2026年1月** 创立，从最初的 3 位成员发展到如今的 **7 人团队**。我们不甘于千篇一律的高中生活——在刷题与考试的间隙，选择用代码与世界对话，用创意为青春留下独特的注脚。

小组的名字来自河北新高考的选科规则，我们想在“物化生”的“主业”之外，让日子更有趣些。

> *&ldquo;学习信息技术，创造美好生活。&rdquo;*
